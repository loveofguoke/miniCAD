# miniCAD
A project for JAD22fall
### Requirements
做一个简单的绘图工具，以CAD的方式操作，能放置直线、矩形、圆和文字，能选中图形，修改参数，如颜色等，能拖动图形和调整大小，可以保存和恢复

### How to Run
VSCode
